# 프로젝트 소개

오늘은 무엇을 먹을지 고민하는 분들을 위한 사용자 맞춤형 음식 추천 어플입니다.

이 앱은 사용자의 지역에서 실시간으로 제공되는 날씨 정보와 개인의 음식 선호도를 바탕으로 가중치를 조정하여 최적의 음식을 추천합니다. 또한, 추천된 음식에 맞춰 근처 식당 정보도 함께 제공하여 편리한 외식 옵션을 제시합니다.
